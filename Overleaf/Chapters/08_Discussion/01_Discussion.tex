\section{The Sensitivity--Reproducibility Trade-off}
% THE CORE OF THE DISCUSSION.
% - The near-separation operating point (\cref{Sensor Structure & the
%   Small-Strain Operating Point}) is WHAT GIVES the high gauge factor
%   (~500, two orders of magnitude over foil gauges) AND what makes the
%   baseline resistance hyper-sensitive to fabrication, temperature, and
%   drift. Frame as a fundamental trade-off, not a process failure.
% - Consequence 1 (optimisation): print-to-print variation can swamp the
%   effect of a single parameter change, so attributing an improvement to
%   a given print setting is hard (link to the 5-stage optimisation,
%   \cref{fig:opt_pla}).
% - Consequence 2 (reproducibility / trust): each bridge has its own
%   baseline and offset, so it never nulls exactly -> residual temperature
%   sensitivity remains (\cref{Wheatstone Half-Bridge Design}; the known
%   3D-print bridge-matching difficulty, \cite{3DprintWheatstoneStrain}).
% - Consequence 3 (productisation): cannot ship a single factory-calibrated
%   unit; each node needs per-unit calibration/commissioning. Note this is
%   normal in the sensor industry, and that process control / sensor
%   binning is a plausible mitigation -> a limitation, not a dead end.
% - Tie back explicitly to the equipment available: more could likely be
%   achieved with better print-process control; reasonable to defer to
%   future work.
