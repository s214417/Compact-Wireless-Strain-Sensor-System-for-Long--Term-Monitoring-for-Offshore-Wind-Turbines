\section{Full System Implementation}
A full system implementation was tested with the Wheatstone bridge print-optimization used in \cref{Wheatstone Half-Bridge Design}, using the assembled final PCB design described in \cref{sec:final_pcb} to acquire data via ZHA. The MCU was powered via the external battery which was connected to another Keithley 2450 Sourcemeter to measure the battery input voltage. The test setup including the Linear stage actuator, stepper motor, stepper driver and Arduino UNO, was placed together with the bias sensors, battery and PCB in a climate chamber, and was tested at a constant temperature of \SI{25}{\celsius} and a constant humidity of 30 \%. The linear actuator was not activated during the test but was used to fasten the sensors to an optimum stretch to optimize the Wheatstone bridge stability. The results can be seen on \cref{fig:FS_ADCogVout} which shows measurement of the Wheatstone bridge, done by both the Keithley 2450 and the on-board IADC.
