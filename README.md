# Compact-Wireless-Strain-Sensor-System-for-Long--Term-Monitoring-for-Offshore-Wind-Turbines
Master’s thesis project covering development of a 3D-printed force sensor and a PCB-based embedded system using the EFR32MG22E MCU with Zigbee communication to Home Assistant. Includes firmware (VS Code + Simplicity Studio), hardware design files, and Overleaf LaTeX thesis documentation.
